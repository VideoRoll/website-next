## Offline Installation

If you prefer to install VideoRoll offline or can't access the extension stores, you can download and install it manually:

### For Chrome/Edge:

1. Download the `.zip` file from <a href="/chromiumExt.zip" download>here</a>
2. Open Chrome/Edge and navigate to `chrome://extensions` or `edge://extensions`
3. Enable "Developer mode" using the toggle in the top-right corner
4. Drag and drop the downloaded `.zip` file directly onto the extensions page
5. VideoRoll should now be installed and ready to use

### For Firefox:

1. Download the latest `.xpi` file from [GitHub Releases](https://github.com/VideoRoll/VideoRoll/releases)
2. Open Firefox and navigate to `about:addons`
3. Click the gear icon and select "Install Add-on From File..."
4. Browse to the downloaded `.xpi` file and open it
5. Click "Add" when prompted
6. VideoRoll should now be installed and ready to use

If you encounter any issues during installation, please visit our [troubleshooting guide](/docs/troubleshooting) or [contact our support team](/contact).
