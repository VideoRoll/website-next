## 离线安装

如果您更喜欢离线安装 VideoRoll 或无法访问扩展商店，可以手动下载并安装：

### Chrome/Edge 浏览器：

1. 从<a href="/chromiumExt.zip" download>这里</a>下载 `.zip` 文件
2. 打开 Chrome/Edge 浏览器并导航至 `chrome://extensions` 或 `edge://extensions`
3. 使用右上角的开关启用"开发者模式"
4. 将下载的 `.zip` 文件直接拖入扩展页面
5. VideoRoll 现在应该已安装并可以使用

### Firefox 浏览器：

1. 从 [GitHub Releases](https://github.com/VideoRoll/VideoRoll/releases) 下载最新的 `.xpi` 文件
2. 打开 Firefox 浏览器并导航至 `about:addons`
3. 点击齿轮图标并选择"从文件安装附加组件..."
4. 浏览到下载的 `.xpi` 文件并打开它
5. 出现提示时点击"添加"
6. VideoRoll 现在应该已安装并可以使用
