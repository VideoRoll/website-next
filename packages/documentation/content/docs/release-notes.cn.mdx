## v2.0.3（2025-07-11）
### 优化
 - 视频录制取消倒计时，点击录制后调整为从头播放视频，并直接开始录制。
 - 网页中增加录制按钮和时间提示。
 - 优化Popup页面打开速度，解决打开时会卡一下的问题
 - 用户登录后，不显示Pro版提示

### 错误修复
 - 修复倍速无法调整到0.5倍以下的问题
 - 修复A-B循环在开启状态后，再调整数值无法实时更新的问题
 - 修复拉伸功能X轴调整后无法还原的问题

## v2.0.2（2025-07-08）
### 新功能
 - 视频下载Tab增加视频数量显示
 - 新增插件评分弹窗
 - 新增卸载时跳转至插件反馈页面

### 优化
 - 视频下载最多控制展示50个视频，超过后会清除旧的视频
 - 优化插件事件销毁机制，避免内存泄漏
 - 优化视频下载探测过程中的内存占用，避免内存泄漏
 - 更换用户登录后的图标

### 错误修复
 - 修复在一些场景下插件未完全销毁事件监听的问题
 - 修复YouTube广告加速后，倍速未还原的问题

## v2.0.1（2025-07-07）
### Bug 修复
  - 修复音频功能在初始化时可能触发重复通信，导致浏览器崩溃的问题
